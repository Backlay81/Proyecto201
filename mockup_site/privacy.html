Privacy & Data Use

This project is for internal research purposes only. No user data is collected via this public website and no interactive data is stored.

Data sources

The system uses the Google Ads API and other public sources for research. All calls to Google APIs are executed from our backend servers and API keys are never exposed in client-side code.

Data retention

Research outputs used internally may be stored as markdown or CSV files for up to 48 hours and then archived; no personal user data is retained.
